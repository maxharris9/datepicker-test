# Datepicker Test
This app demonstrates the maxharris9:datepicker package.

## Installation

	git clone https://github.com/maxharris9/maxharris9-datepicker.git
	git clone https://github.com/maxharris9/maxharris9-datepicker.git
	cd datepicker-test/packages/
	ln -s ../../maxharris9:datepicker .	
	
## Running

	cd datepicker-test
	meteor
